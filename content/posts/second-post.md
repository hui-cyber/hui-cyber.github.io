## 为什么选择 Hugo

Hugo 速度快、无运行时依赖，适合个人博客长期维护。

## 初始化项目

- 创建仓库
- 添加 PaperMod 主题
- 配置 `hugo.toml`

## 内容结构

- `content/posts/`：长文文章
- `content/notes/`：短记
- `content/projects/`：项目展示

## 发布流程

- GitHub Actions 构建
- 自动发布到 GitHub Pages
